TrailheadZ

This is my first ever project I worked on with a group. I'm very proud of the work put into this and look forward to working with others on similar projects!

TrailheadZ is a web application that takes advantage of a couple different APIs. The first being REI's Hiking Project API and Google's own Maps and Geolocation API.

Another key feature that we wanted to include was a way for veteran and new hikers to connect with each other. With knowledge of JSON, we were able to use Reddit's Colorado Hikers subreddit to implement a similar feature.
